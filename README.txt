# Introduction

This repository contains the source code for a React-based web application designed to manage user interactions related to waste management and environmental education. The application includes features such as user authentication, profile management, and educational resources about different types of waste and waste management practices.

## Features

- **User Authentication**: Allows users to sign up, log in, and manage their profiles.
- **Educational Resources**: Offers information on various types of waste, including liquid, solid, organic, recyclable, and hazardous waste.
- **Interactive UI**: Utilizes modals, accordions, and popovers to provide a dynamic user experience.
- **Responsive Design**: Ensures that the application is usable on both desktop and mobile devices.

## File Structure

- `App7.js`: Handles user signup and form validations.
- `authority_nav6.js`, `nav3.js`, `nav4.js`, `nav5.js`: Provide navigation and additional user interface functionalities like modals for different user actions.
- `form_proj.js`, `signup.js`, `signupform.js`: Related to user signup and authentication processes.
- `sql3.js`: Backend script for handling SQL database interactions.
- `What.js`, `Why.js`: Present educational content about waste management.
- `profile.js`: Manages user profile fetching and displaying.
- `CustomNavbar.jsx`, `home.jsx`, `about.jsx`, `nav2.jsx`, `news.jsx`: React components for various parts of the application including the home page, about page, and custom navigation bar.

## Setup

To run this application, you will need Node.js and npm installed on your computer. After cloning the repository, run the following commands to start the application:

```bash
npm install     # Installs all the necessary dependencies
npm start       # Starts the application on localhost:3000
